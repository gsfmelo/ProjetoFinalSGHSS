Projeto Multidisciplinar Final do Curso de Análise e Desenvolvimento de Sistemas do Centro Universitário Internacional (UNINTER) realizado pela aluna Geovanna Silva Ferreira de Melo, RU: 4466904. 
O projeto tem como objetivo lidar com um Sistema de Gestão Hospitalar e de Serviços de Saúde (SGHSS) com ênfase na Qualidade de Software. 
O programa consiste em códigos de simulação para testes de software.
